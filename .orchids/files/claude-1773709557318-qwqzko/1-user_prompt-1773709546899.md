# PROJECT CONTEXT

This is **Vezo (formerly Mezo Market)** — a veNFT marketplace for:

• veBTC
• veMEZO

Users can:

• list NFTs
• buy NFTs
• view intrinsic value
• view discounts
• track activity

The system is **fully functional and deployed-ready**. here is more detailed context "Problem

Mezo’s dual-token economic system (veBTC + veMEZO) creates locked positions represented as NFTs. These veNFTs carry intrinsic value. Locked BTC or MEZO with associated voting power, fee claims, and boost potential. However, the community currently lacks a dedicated marketplace for trading these positions.

Users who want to exit a locked position early, acquire voting power without locking tokens themselves, or find discounted veNFT positions have no streamlined way to do so. This limits capital efficiency and creates friction for users seeking liquidity or strategic positioning within Mezo Earn.

Similar ecosystems (Velodrome/Aerodrome) have solved this problem with veNFT marketplaces like OpenXSwap and Vexy, which enable users to list, buy, and sell veAERO/veVELO locks at market-determined prices.

---

Proposed Solution

Fork and adapt an existing, already-audited veNFT marketplace for Mezo's veBTC and veMEZO NFT standards.

This bounty requires using audited smart contract code as the foundation. Building marketplace contracts from scratch is not within scope. The goal is to minimize new attack surface and leverage battle-tested code that has already secured significant value on other chains.

The adapted marketplace must enable Mezo users to:

* List veNFTs for sale: Set asking prices (in BTC, MEZO and MUSD) for veBTC and veMEZO positions
* Browse available listings: View all listed veNFTs with key metadata (locked amount, lock expiry, current voting weight, discount rate vs. spot price)
* Purchase veNFTs: Buy listed positions with gas estimation and confirmation flows
* Cancel listings: Delist unsold veNFTs at any time
* View position details: Display intrinsic value calculations (locked tokens × spot price), time remaining, voting power decay, and price-to-value ratios

Bonus Features:

* Transaction/Sale history per veNFT
* Price alerts / watchlist functionality
* Analytics dashboard (floor prices, volume, average discounts)
